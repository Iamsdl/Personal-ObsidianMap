More keywords that allow you to control whether child classes have access to parent members. They are:
- `protected` - members can be accessed from the owner class or inheriting classes.
- `protected internal` - members can be accessed from the owner class, inheriting classes, and other classes that are declared in the same [[Project|project]]
- `private protected` - members can be accessed from the owner class and from inheriting classes that are declared in the same project
