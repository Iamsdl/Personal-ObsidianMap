In layman's terms, programming means
> **A [[#Programmer|person]] [[#Programming|giving instructions]] to the computer [[#Programming language|in a language]] that it can understand.**
> The result is called [[#Program|program]].
# Programmer
A programmer is the one [[#Programming|programming]].
# Programming
Programming (also called coding) means [[What does programming even mean|program]] writing. It is done using a [[#Programming language|programming language]].
# Programming language
The programming language (also called coding language) is the [[Set|set]] of all available [[Instructions (statements)|instructions]] with which a program can be written.
There are many languages to choose from, but their type is more important:
## High level languages
Those are closer to human languages. Assuming a cleanly written program, the behavior can be understood just by reading the code.
Popular examples:
- C#
- Java
- Python
- TypeScript
- JavaScript
- ...and many others
## Low level languages
Those are closer to machine language. Hard to read by humans, with abbreviated names and without the complex features of higher level languages.
Assembly is one such language.
# Program
A program is an [[Algorithm|algorithm]] that can be [[Execution|executed by a machine]].
Parts of the program or the whole program itself may be referred to as "the code".
