These operators are a combination of [[Basic operations#Arithmetic operations|arithmetic operations]] and [[Data and Variables#Assignment|variable assignment]].
They are

- increment `++` -> `x++` is equivalent to `x = x + 1`
- decrement `--` -> `x--` is equivalent to `x = x - 1`
- increment by `+=` -> `x+=2` is equivalent to `x = x + 2`
- decrement by `-=` -> `x-=2` is equivalent to `x = x - 2`
